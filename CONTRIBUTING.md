# Contributing

## Contact and support

- For bugs, enhancement ideas, or usage questions, use [GitHub Issues](https://github.com/Ci303/Invoke-WindowsCleanup/issues).
- For security concerns, follow [SECURITY.md](./SECURITY.md).

## Quick contribution process

1. Fork the repository.
2. Create a feature branch.
3. Make small, focused changes.
4. Validate the script change logically in a test environment.
5. Open a pull request with a clear summary of what changed and why.

## Scope

Keep this repository focused on Windows PowerShell helpers for cleanup and tray-icon maintenance.
